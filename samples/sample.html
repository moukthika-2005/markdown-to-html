Markdown to HTML Converter

This is a simple Markdown to HTML converter.

Features

  • Convert headers
  • Bold and italic text
  • Lists
  • Links
  • Inline code: print("Hello")

Code

  1. bash code
npm install
npm run dev
  1. Python code
def greet():
    print("Hello World")
  1. Java code
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
